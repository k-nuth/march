march
=========

Knuth Microarchitecture Management (CPUID)

Getting started
---------------

Stable version:

.. code-block:: bash

    $ pip install --upgrade march

Development version:

.. code-block:: bash

    $ pip install --upgrade --index-url https://test.pypi.org/pypi/ march
